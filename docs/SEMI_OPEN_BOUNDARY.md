# Semi-Open Boundary

The public-preview repository is a controlled public surface, not a disclosure
of the full YIYUAN-ASSETS core.

## Public Surface

- public documentation;
- public-preview contracts;
- selected examples;
- safe issue templates;
- contribution and support boundaries;
- non-sensitive governance evidence.

## Protected Surface

- core graph authority implementation;
- graph write governance internals;
- commercial entitlement, billing, and anti-abuse logic;
- private security reports and exploit details;
- internal operational runbooks;
- secrets, tenant data, customer data, and provider transcripts.

## Sync Principle

Public materials should be generated, copied, or curated from the private core
only through an explicit review boundary. The public repository must not inherit
private Git history.
