# Support

This repository provides public-preview, best-effort support only.

Appropriate public support topics:

- documentation clarity;
- public-preview evaluation questions;
- public API or example feedback;
- non-sensitive integration questions;
- issue template or contribution process questions.

Do not use public support for:

- production incidents;
- private vulnerability details;
- secrets or credentials;
- tenant, customer, or personal data;
- billing, refund, legal, tax, trademark, or regional compliance questions;
- protected core internals or proprietary orchestration.

Public support signals are candidate feedback. They do not mutate canonical
system truth, internal graph authority, release gates, entitlement boundaries,
or commercial commitments.
