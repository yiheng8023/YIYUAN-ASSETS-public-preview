# Contributing

Thank you for helping evaluate YIYUAN-ASSETS during public preview.

## Safe Contribution Areas

- documentation clarity;
- public-preview examples;
- public API contract feedback;
- typo fixes;
- issue template improvements;
- non-sensitive integration questions;
- public-safe bug reports with reproduction details.

## Areas Requiring Maintainer Direction

- license changes;
- security-sensitive reports;
- protected core behavior;
- graph write authority;
- commercial entitlement, billing, or anti-abuse logic;
- production deployment promises;
- large architectural changes.

## Do Not Submit

- secrets or credentials;
- customer, tenant, or personal data;
- exploit details in public issues;
- proprietary business logic that you cannot share;
- copied third-party material without permission.

Contributions are reviewed as public-preview candidate signals. A merged change
does not imply production support, commercial authorization, or canonical
internal graph authority.
